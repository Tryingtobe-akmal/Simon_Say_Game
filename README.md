# Simon Say Game

A simple and fun Simon Say memory game built with HTML, CSS, and JavaScript.

## About This Project

This project is a browser-based memory game where the player has to remember the color sequence shown by the game and repeat it correctly. With every correct round, the level increases and the sequence becomes longer.

## My Work

- I wrote the HTML structure of the game.
- I wrote the JavaScript logic for the game.
- The styling in `style.css` was added with help from Codex.

## Features

- Random color sequence generation
- Level-based gameplay
- User click detection
- Game-over message
- Score display
- Highest score display during the current session
- Responsive and modern CSS styling

## How To Play

1. Open the game in a browser.
2. Press any key to start the game.
3. Watch the color that flashes.
4. Click the same color.
5. In the next level, repeat the full sequence from the beginning.
6. If you click the wrong color, the game ends and your score is shown.

## How To Run Locally

Download or clone this project, then open `index.html` in any browser.

## GitHub Pages

This game can be hosted using GitHub Pages because it only uses HTML, CSS, and JavaScript.

After uploading the project to GitHub:

1. Go to the repository settings.
2. Open the Pages section.
3. Select the `main` branch.
4. Select the `/root` folder.
5. Save the settings.

GitHub will create a website link that friends can open to play the game.
